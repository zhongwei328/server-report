五、腾讯云控制台操作指引

5.1 找公网 IP

  1. 登录 腾讯云控制台
  2. 顶部搜索「云服务器」→ 进入 CVM 列表
  3. 找到你的服务器，IP 地址列就是公网 IP

5.2 开放端口（安全组）

  1. 云服务器列表 → 点击服务器名称 → 「安全组」
  2. 点击安全组名称 → 「入站规则」→ 「添加规则」
  3. 填写：类型=自定义，来源=0.0.0.0/0，端口=22，协议=TCP
  4. 保存

5.3 重启服务器

  1. 云服务器列表 → 勾选服务器
  2. 顶部「重启」按钮（不要经常重启）

六、连接服务器的两种方式

💻 方式一：SSH 终端（推荐）

适合：日常操作、跑命令、长期使用

# macOS / Linux 终端
ssh ubuntu@你的公网IP

# Windows 用 PowerShell
ssh ubuntu@你的公网IP

🌐 方式二：腾讯云「登录」按钮

适合：偶尔看一眼、不想开终端

云服务器控制台 → 点击「登录」→ 选择「一键登录」或「密码登录」

八、下一步建议

1️⃣ 接入飞书

在飞书开放平台创建 Bot，配置 App ID 和 Secret，实现手机消息控制服务器。参考教程第八节。

2️⃣ 搭建三 Agent 协作

用 tmux + Hermes Profiles 启动三个常驻 Agent（指挥官、研究员、质检员），互相监督辅助。

3️⃣ 开通 HTTPS 回调

在腾讯云安全组开放 443 端口，购买域名或用已有域名，配置飞书 Bot 回调地址。

4️⃣ 定时备份

设置 cron 任务定期备份 ~/.hermes/ 目录到腾讯云硬盘或对象存储。

📌 小结：你的服务器配置均衡（2核/3.6G内存/59G盘），跑 Hermes Agent 绑绑有余。核心限制是没有公网 IP（只有内网 10.8.0.9），飞书 Bot 回调需要另外处理。
